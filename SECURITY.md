# 🔒 Kebijakan Keamanan GDIPS

## Versi yang Didukung

| Versi | Status |
|-------|--------|
| 2.1.x | ✅ Didukung |
| 2.0.x | ❌ Tidak didukung |
| < 2.0 | ❌ Tidak didukung |

## Melaporkan Vulnerability

Jika kamu menemukan kerentanan keamanan, **JANGAN buka public issue.**

Laporkan secara private melalui:

1. **Email:** fless@gdi.ps
2. **Discord DM:** @fless (di [server Discord GDIPS](https://discord.gg/YyeZ2Sxjgf))
3. **GitHub Security:** Gunakan [GitHub Private Reporting](https://github.com/flessan/GDIPS/security/advisories/new)

### Format Laporan

```
Judul: [SECURITY] Deskripsi singkat vulnerability

Severity: Critical / High / Medium / Low
Versi terpengaruh: x.x.x
Komponen: (contoh: login.php, upload endpoint)

Deskripsi:
...

Langkah reproduce:
1. ...
2. ...

Dampak:
...

Usulan perbaikan (opsional):
...
```

## Respons Kami

- **Acknowledgment:** Dalam 48 jam
- **Initial assessment:** Dalam 7 hari
- **Fix timeline:** Tergantung severity
  - Critical: 1-3 hari
  - High: 3-7 hari
  - Medium: 1-2 minggu
  - Low: Sesuai roadmap
