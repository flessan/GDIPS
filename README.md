## 🌱 Why GDIPS

**More than a server — a collaborative experiment.** Built on the belief that:

- 🔓 **Open-source first** — every line of code is transparent and accessible
- 🤝 **Anyone can contribute** — developers, designers, testers, idea-givers
- 📚 **Learning by building** — grow your skills alongside real projects
- 🌍 **Community ownership** — this server belongs to its contributors

> *"Code is poetry when written together. GDIPS is our shared canvas."*

## 📜 Contribution Guidelines

1. **Respect the Codebase**  
   - Follow existing code style and structure
   - Write clear commit messages

2. **Collaborate Openly**  
   - Discuss big changes in issues first
   - Welcome feedback and iterations

3. **Document Your Work**  
   - Comment complex logic
   - Update relevant documentation

4. **Be Patient & Kind**  
   - Reviews may take time (we're volunteers!)
   - Constructive criticism only
